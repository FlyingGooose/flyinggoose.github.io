AI遭遇战定义

各地点遭遇战的生成

data\ailife\encounters\encountersettings.scr

  1. Encounter(s) - 这是遭遇类别的声明，s是一个唯一的遭遇类别名称。

  2. Pool(s) - 这是池的名称，通常在AIPresetPool.scr文件中定义。池用于存储可用的遭遇内容。

  3. CooldownTime(f) - 这是遭遇之间的激活冷却时间，以秒为单位。

  4. CooldownTimePendingActive - 这是在遭遇处于待激活状态时的冷却时间，以秒为单位。

  5. SaveCooldownTime(b) - 如果设置为true，表示冷却时间应该被保存，否则设置为false。这个参数可能用于控制冷却时间的持久性。

  6. MaxActiveInstances(i) - 这是遭遇类别实例的最大数量限制。

  7. TimeOfDay(f, f) - 这是遭遇可以激活的小时范围。两个浮点数参数表示开始时间和结束时间，通常以小时为单位。

  8. EncounterGroup(s) - 这是遭遇的组别，通常是一个字符串，用于将遭遇分类到不同的组中。默认值是”Default”。

  9. WeatherEncounterCondition(i, i) - 这是可选的天气条件，用于限制在特定天气条件下激活遭遇。两个整数参数表示最小和最大的天气条件，通常以数字或特定的字符串表示不同的天气类型。

  10. ExpRewardMultiplier(f) - 这是用于遭遇经验奖励的倍数。它可以影响遭遇的难度和经验奖励的计算。

    Encounter(“Encounter_Template”)
    {
    Difficulty(“Easy”)
    {
    Pool(“Encounter_Template”); // 在”Easy”难度下，使用名为”Encounter_Template”的池
    }

    Difficulty(“Medium”)
    {
    Pool(“Encounter_Template”); // 在”Medium”难度下，仍然使用名为”Encounter_Template”的池
    ExpRewardMultiplier(2.0); // 在这个难度下，将经验奖励乘数设置为2.0
    }

    Difficulty(“Hard”)
    {
    Pool(“Encounter_Template”); // 在”Hard”难度下，仍然使用名为”Encounter_Template”的池
    ExpRewardMultiplier(3.0); // 在这个难度下，将经验奖励乘数设置为3.0
    }

    CooldownTime(1800.0); // 设置遭遇之间的冷却时间为1800秒（30分钟）
    TimeOfDay(-1, -1); // 遭遇可以在任何时间激活，因为开始和结束时间都为-1
    }

其中EncounterGroup定义在 data\ailife\encounters\encountermanagerconfig.scr中

AI池定义

池中用来描述相关的ai生成，常常被遭遇战中调用

POOL池文件全部定义在data\scripts\aipresetpool.scr中

如Pool(“Encounter_Template”);

AI生成的区域

data1.13\ailife\areasettings.scr

用来设置特定区域中的ai生成，可控制生成个数，非遭遇战。

  1. AreaSettings("<attribute name>") - 这是一个示例，用于定义特定区域的设置。区域通常与属性相关联，用于控制游戏中任务、事件或AI生成等方面的行为。
  2. EncounterDifficulty("Not_set") - 这个参数用于设置特定区域内的遭遇难度。它可以有四个值： “Not set”、”Easy”、”Medium” 和 “Hard”，用于表示遭遇的难度级别。
  3. EncounterCategoryWeight("EncounterCategory,10") - 这个参数用于设置特定区域中遭遇类别的权重。它允许为多个遭遇类别分配权重，以控制它们在区域内的出现频率。在示例中， “EncounterCategory” 表示遭遇类别的名称，而 “10” 表示权重。
  4. DynamicSpawnerDayPresetPool("Day_Exterior") - 这个参数指定了在白天使用的动态生成器（Dynamic Spawner）的预设池（Preset Pool）的名称。动态生成器通常用于在游戏中生成AI或其他游戏元素。
  5. DynamicSpawnerNightPresetPool("Night_Exterior") - 与上述相似，但是用于夜间时使用的动态生成器的预设池。
  6. DynamicSpawnerDayDensity("Easy") - 这个参数用于设置白天时从动态生成器生成的AI的密度。它可以有五个值： “None”、”Easy”、”Medium”、”Hard” 和 “VeryHard”，表示生成的AI的数量和难度级别。
  7. DynamicSpawnerNightDensity("Hard") - 与上述相似，但是用于夜间时从动态生成器生成的AI的密度。
  8. HumanReinforcementDataId("PK_Area_01") - 这个参数用于指定由人类召唤者在街道上生成的AI组的组成数据。它通常用于描述召唤者生成的敌人或帮手。
  9. InfectedReinforcementDataId("DZ_01") - 这个参数用于指定由大声噪音或尖叫声在黑暗区域（dark zones）生成的感染者组的组成数据。通常用于描述由噪音或事件引起的感染者生成。
